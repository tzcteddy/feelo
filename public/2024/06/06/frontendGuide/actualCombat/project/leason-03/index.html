Axios 安装与使用

首先，我们在vue项目调用接口一般用 Axios 库实现，详细API可在官网查看。

Axios 是一个基于 Promise 的 HTTP 客户端，用于浏览器和 Node.js，在浏览端中使用 XMLHttpRequests实现。它支持多种请求方式，如 GET、POST、PUT、DELETE 等，并且可以处理各种类型的请求和响应数据。
